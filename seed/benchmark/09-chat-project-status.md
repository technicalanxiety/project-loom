# Project Sentinel status

Project Sentinel has the status of "in flight." The project kicked
off at the start of the quarter; the discovery phase is complete,
the architecture review has been signed off, and the first slice
of implementation work is underway in the platform team's backlog.

The current week's work on Project Sentinel covers two tracks:
the data-plane changes that the architecture review pulled forward,
and the dashboard surface that operators will use to monitor the
new pipeline. Project Sentinel has a status check every Tuesday;
the meeting notes are kept in the team's working doc and surfaced
in the project review on the last Thursday of each month.

Risks for Project Sentinel right now: one external dependency on
the data team's warehouse migration is the long pole, and a
hiring delay has put the second engineer's start date about two
weeks later than the plan assumed. Neither risk is in the red;
both are tracked on the status doc and discussed at the Tuesday
check-in.
